Architecture

How Trust Works

Register

Agent gets on-chain identity (ERC-8004)

Bond

Stake value as commitment (AI Partnership Bonds)

Verify

Any agent can check trust on-chain

Score

Street Cred: 0–95 composite score

Street Cred Breakdown

Identity30 pts
Has Bond25 pts
Active Bond15 pts
Bond Tier20 pts
Multiple Bonds5 pts
🥉 Bronze 20+
🥈 Silver 40+
🥇 Gold 60+
💎 Platinum 80+
Philosophy

Why Bonds > Scores

Feature Vaultfire Bonds Trust Scores
Skin in the game ✓ Real economic stake ✗ None
Manipulation resistant ✓ Economic cost to fake ✗ Easy to game
Verifiable ✓ On-chain, trustless ✗ Centralized API
Cross-chain ✓ 4 chains, day one ✗ Single provider
Accountability ✓ Bond at risk on failure ✗ Score drops
